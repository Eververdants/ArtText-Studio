## [1.0.0] - 2026-01-25

### 新增 ✨
- 🎨 AI 智能文字艺术生成功能
- 🤖 集成 Google Gemini AI 进行风格分析和背景生成
- 🖼️ 40+ 精心设计的预设样式
- ✍️ 11 种精选字体（包括中英文字体）
- 📐 5 种专业布局模式
- 🎛️ 深度参数调节系统
  - 字体大小和行高调节
  - 阴影和描边效果
  - 背景图片调节（亮度、模糊、对比度等）
  - 纸张质感效果
- 🌍 中英文双语界面
- 📱 响应式设计，支持移动端
- 💾 高清导出（3x 分辨率）
- 📋 一键复制到剪贴板
- 🎨 自定义背景图片上传
- 🔄 随机风格生成

### 技术特性 🛠️
- ⚡ 使用 Vite 6.2 构建，启动速度快
- ⚛️ React 19.2.3 + TypeScript 5.8
- 🎨 Tailwind CSS 样式系统
- 🚀 GitHub Actions 自动部署
- 📦 代码分割优化
- 🔒 环境变量安全管理

### 文档 📚
- 📖 完整的 README 文档
- 🚀 快速开始指南
- 📝 详细的部署指南
- 🤝 贡献指南
- 📋 Issue 和 PR 模板

### 部署 🌐
- ✅ GitHub Pages 自动部署配置
- 🔧 部署脚本（支持 Windows 和 Unix）
- 🔐 环境变量配置示例
